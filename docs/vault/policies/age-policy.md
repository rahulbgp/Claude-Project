# Age Eligibility Policy

**Tags:** #policy #eligibility #age

## Rule

| Criteria | Range |
|----------|-------|
| Eligible age | 21 – 60 years |
| Below 21 | **NOT ELIGIBLE** |
| Above 60 | **NOT ELIGIBLE** |

## Implementation

Checked by [[agents/eligibility-checker]] via `check_age_eligibility` tool.
